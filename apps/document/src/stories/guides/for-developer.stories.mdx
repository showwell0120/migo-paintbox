# For Developers

## 設定 registry

設定完才能 install Paintbox 的套件。

1. 建立 [personal access token](https://docs.gitlab.com/ee/user/profile/personal_access_tokens.html#create-a-personal-access-token)，`scope` 勾選 `api` 即可
2. 設定 npm config－

```bash
# 設定 registry 路徑
npm config set @paintbox:registry https://gitlab.migoinc.com/api/v4/projects/319/packages/npm/

# 設定 auth token
npm config set -- 'https://gitlab.migoinc.com/api/v4/projects/319/packages/npm/:_authToken' "<your_token>"

# 確認相關設定是否正確
npm config list
```

#### 我要使用哪個 package？

✅ 幾乎都會使用到的話：`@paintbox/react-ui`

✅ 只需要套基本的 UI 的話：`@paintbox/react-foundation`

✅ 只會用到表格相關的元件的話：`@paintbox/react-table`

✅ 使用者輸入相關元件有用到三種以上的話：`@paintbox/react-form`，否則安裝特定套件即可，例如 `@paintbox/react-button`。

✅ 資料顯示相關元件有用到三種以上的話：`@paintbox/react-display`，否則安裝特定套件即可，例如 `@paintbox/react-tag`。

✅ feedpack 相關元件有用到三種以上的話：`@paintbox/react-feedpack`，否則安裝特定套件即可，例如 `@paintbox/react-spinner`。
