# About

Paintbox 是 Migo Product Team 中 Web 前端共用的元件和函式庫，並且以 React/TypeScript 作為主要的實現方式。

## 快速上手

這個平台是使用 [Storybook](https://storybook.js.org/) 建置，各元件的文件和 use cases 都叫做 **story**，並且遵循其提供的格式撰寫。

左方側欄是主選單，目前的目錄章節為－
- **GUIDES**：給不同的角色看的導覽
- **PROJECT**：: 建置前端專案的步驟
- **REACT**：React 元件的 stories
- **MODEL**：data model 的 stories
(陸續新增中)

展開後會依照分類列出底下元件的 story。點擊某個 story 後，可以在主畫面的右上 tab 選擇其中一種預覽模式-
- **Canvas**：預設，只會顯示 use cases。下方會根據外掛設定顯示不同的控制面板，並且可以對 use cases 進行進一步的操作
- **Docs**：不僅顯示 use cases，也會顯示說明(如果有的話)。不過這個模式只會靜態顯示
