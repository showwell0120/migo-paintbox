# For Designers

>- Paintbox 會集中 Design Foundation 以及產品中可獨立出來的 UI 元件供其他專案開發上使用。
>
>- 為了跟 Designers 和 Developers 有良好的溝通，因此選擇 Storybook 作為元件庫的視覺化和文件化。
>
>- 目前的 UI 元件都是以 React 實作，因此相關的文件和 use cases 都會集中在 **REACT** 的選單中。

## 各分類的說明

### Foundation

集合了 Design Foundation 中所有的項目（除了 Components），包含了配色、排版、圖檔等相關的元件。

### Table

集合了跟表格相關的元件，例如：THead、SearchBar、Pagination 等。

### Form

集合了跟使用者輸入相關的元件，例如：Button、CurrencyInput 等。

### Display

集合了跟資料呈現相關的元件，例如：Tag、Chip、TreeView 等。

### Feedback

集合了因狀態或操作的改變而顯示，並且有時需要讓使用者進行動作的元件，例如：Toast、Modal 等。
