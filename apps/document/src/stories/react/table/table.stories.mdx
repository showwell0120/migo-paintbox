# @paintbox/react-table

---

## Normal Style 

> 一般樣式表格的元件集合。需搭配 `@tankstack/react-table` 傳入各元件需要的 props，提供以下共同的 props－

```javascript
export interface NormalTableProps {
  children: React.ReactNode;
  className?: string;
  onClick?: () => void;
}
```

## Preview Style 

> 預覽樣式表格的元件集合。需搭配 `@tankstack/react-table` 傳入各元件需要的 props，提供以下共同的 props－

```javascript
export interface PreviewTableProps {
  children: React.ReactNode;
  className?: string;
}
```
