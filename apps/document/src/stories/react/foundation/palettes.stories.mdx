# Palettes

> Palettes 以 SCSS 產生一系列的顏色變數和字型設定。

## Mockup
- [Figma](https://www.figma.com/file/Sv8VnmaINqyDQcpdO3cz5M/%F0%9F%96%A5-Foundation?node-id=5%3A2)


## Usage

### main.scss

`main.scss` 主要是有關顏色、字體、基本樣式的統一定義。需要在專案樣式檔的開頭匯入。

```css
/* index.scss */
@import '~@paintbox/react-foundation/palettes/main.scss';

.data-list-price {
    color: var(--text-muted);
}
```
