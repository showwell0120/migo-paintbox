# Palettes

> Palettes 以 SCSS 產生一系列的顏色變數和 class 名稱，並且提供 utility 動態取得樣式。

## Usage

目前在 Palettes 中共有兩個 SCSS 檔案

### main.scss

`main.scss` 主要是有關顏色、字體、基本樣式的統一定義，以及產生公用的 class 名稱。因此需要在專案樣式檔的開頭匯入。

```css
/* index.scss */
@import '~@paintbox/react-foundation/palettes/main.scss';

.data-list-price {
    color: var(--text-muted);
}
```

當匯入這個檔案時會進行以下初始化－
1. 匯入 Roboto Font
2. 在 `*` 下設定字體和 box 相關的樣式
3. 定義 `$palettes` 
4. 在 `:root, body` 下，根據 light / dark theme，以 `$palettes` 產生一系列的 CSS 變數
5. 以 `$palettes` 的 `accent-light` 產生一系列 `.migo-filled-xxx` 和`.migo-outlined-xxx` 的 class 名稱
5. 以 `$palettes` 的 `text-light` 產生一系列 `.migo-xxx` 的 class 名稱

### functions.scss

`functions.scss` 定義動態取得樣式的 utility。因為這個檔案有匯入 `main.scss`，如果有使用到這個檔案的話，就不需要再匯入 `main.scss`。

目前有定義 `getColor` 的 utility， 可以用語意化參數的方式，取得 `$palettes` 下的特定顏色。

```css
@import '~@paintbox/react-foundation/palettes/functions.scss';

.data-id {
    color: getColor(gray, 700);
}
```
