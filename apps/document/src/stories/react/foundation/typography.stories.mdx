# Typography

> Typography 定義了各種字體樣式的元件，並且提供 props 設定注入樣式或 class 名稱。

```javascript
export interface TypographyProps {
  className?: string;
  style?: Partial<CSSProperties>;
  children?: ReactNode;
}
```

## Mockup
- [Figma](https://www.figma.com/file/Sv8VnmaINqyDQcpdO3cz5M/%F0%9F%96%A5-Foundation?node-id=9%3A377)

## Usage 

```javascript
import { H1 } from '@paintbox/react-foundation';
return <H1>text content</H1>
```
