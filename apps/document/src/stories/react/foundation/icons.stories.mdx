# Icons

> Icons 集中了目前在專案中有用到的 SVG 圖檔，並且以 React FC 的形式封裝。每個 icon 原件都有提供 props 設定注入樣式。

```javascript
export interface SVGProps {
  size?: number;    // 寬度
  fill?: string;    // 背景的顏色  
  stroke?: string;  // 線條的顏色
}
```

## Mockup

- [Figma](https://www.figma.com/file/Sv8VnmaINqyDQcpdO3cz5M/%F0%9F%96%A5-Foundation?node-id=14%3A2361)

## Usage

```javascript
import { SortUp } from '@paintbox/react-foundation';
return <SortUp />
```
