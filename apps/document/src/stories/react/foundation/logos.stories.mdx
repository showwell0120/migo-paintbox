# Logos

> Logos 集中了所有與 Migo 相關的 Logo 圖檔，並且提供 props 設定注入樣式。

```javascript
export interface SVGProps {
  size?: number;    // 寬度
  fill?: string;    // 背景的顏色  
  stroke?: string;  // 線條的顏色
}
```

## Mockup

目前沒有。

## Usage 

```javascript
import { FilledLogo, SolidLogo } from '@paintbox/react-foundation';
return <FilledLogo />
```
