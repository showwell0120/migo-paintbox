# Icons

> Icons 集中了目前在專案中有用到的 SVG 圖檔，並且以內建的 ReactComponent 的形式匯出。Icon 元件可以設定原生支援的 props。常見的有－

```javascript
export interface SVGProps {
  className?: string | undefined;
  height?: number | string | undefined;
  width?: number | string | undefined;
  fill?: string | undefined;
  fillOpacity?: number | string | undefined;
  stroke?: string | undefined;
  strokeOpacity?: number | string | undefined;
  strokeWidth?: number | string | undefined;
}
```

如果發現設定 `fill` 無法改變填色，可以新增 `className`，並且對元件底下的 `path` element 設定 `fill` 的顏色。

```scss
.fill-path-primary {
    path {
        fill: var(--primary);
    }
}
```

```javascript
<MediaIcons.CameraReelsFill className="fill-path-primary" />
```
