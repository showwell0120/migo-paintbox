# Checkbox

> 核對方塊。以控制元件的方式實作，提供以下 props－

```javascript
export interface SVGProps {
  checked: boolean;                     // current state
  label?: string;                       // checkbox 右方的文字
  onChange: (checked: boolean) => void; // event handler
}
```

## Mockup

- [Billing System on Figma](https://www.figma.com/file/rvTsTJLCarPAEYvdXQuRDY/Billing-System---MVP?node-id=5289%3A11324)

## Usage 

```jsx
import { ReactCheckbox } from '@paintbox/react-checkbox';

export const Sample = () => {
  const [checked, setChecked] = React.useState<boolean>(false);
  return <ReactCheckbox checked={checked} onChange={setChecked} />;
};
```
